Variables in TypeScript

Learn the fundamentals of TypeScript variables, data types, and basic syntax.

📦 What are Variables?

Variables are like labeled boxes that store information. Imagine you have boxes in your room - each box has a label and stores something different. Variables work the same way!

In TypeScript, we create variables using these keywords:

  • let - for things that might change (like your age)
  • const - for things that stay the same (like your name)
  • var - old way (we don't use this anymore)

💡 Think of 'let' as a box you can put new things in, and 'const' as a box that's sealed shut!

🎭 Different Types of Data

TypeScript can store different types of information:

📝 Text (String)

Words and sentences:

const myName = "Luka";

🔢 Numbers

Any number:

let age = 25;

✅ True/False (Boolean)

Yes or no answers:

const isStudent = true;

🤔 Think of it like different types of containers - some hold text, some hold numbers, some hold lists of things!
